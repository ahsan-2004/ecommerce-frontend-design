# Ecommerce Frontend Design

## Overview
This project is part of DevelopersHub internship tasks.  
It is built with React + Vite and aligned with the Ecommerce Web Design Figma.

## Features
- Header with logo, search bar, and navigation
- Footer with links and copyright
- Home page hero section with banner and CTA
- Featured categories grid
- Product listing page with grid and pagination
- Product details page with dropdown, reviews, and related products

## Tech Stack
- React
- Vite
- CSS3
- JavaScript
- GitHub for version control
